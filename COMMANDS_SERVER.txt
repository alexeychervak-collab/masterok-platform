=== STROYKA - Commands for Server ===

1. Clone from GitHub:
cd ~
git clone https://github.com/gahshsfshsh/stroyka-platform.git
cd stroyka-platform

2. Setup Backend:
cd backend
python3.11 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
echo 'DATABASE_URL=sqlite:///./stroyka.db' > .env
cd ..

3. Setup Frontend:
cd landing
npm install
echo 'NEXT_PUBLIC_API_URL=http://localhost:8000' > .env.local
npm run build
cd ..

4. Start Backend:
cd backend
source venv/bin/activate
uvicorn app.main_simple:app --host 0.0.0.0 --port 8000

5. Start Frontend (new terminal):
cd landing
npm start
