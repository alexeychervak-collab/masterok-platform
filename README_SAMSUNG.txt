🚀 БЫСТРАЯ УСТАНОВКА:
─────────────────────────────────────────────────────────────────────

СПОСОБ 1: Через USB-кабель и ADB (АВТОМАТИЧЕСКИ)
   1. Подключите Samsung к компьютеру через USB
   2. Включите "USB отладку" на телефоне
   3. Запустите: БЫСТРАЯ_УСТАНОВКА_НА_SAMSUNG.bat
   4. Готово!

СПОСОБ 2: Вручную (БЕЗ КОМПЬЮТЕРА)
   1. Скопируйте МастерОК-v1.0.4-arm64.apk на телефон
   2. Включите "Установка из неизвестных источников"
   3. Откройте APK файл на телефоне
   4. Нажмите "Установить"
   5. Готово!

🔧 РЕШЕНИЕ ПРОБЛЕМ:
─────────────────────────────────────────────────────────────────────

❌ "Приложение не установлено"
   → Удалите старую версию: Настройки → Приложения → МастерОК → Удалить

❌ "Приложение не открывается"
   → Попробуйте другой APK (arm64 → armv7 или наоборот)
   → Проверьте версию Android (нужна 5.0+)

❌ "Нет подключения к серверу"
   → Это нормально, backend нужно настроить отдельно
