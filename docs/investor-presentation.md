## Слайд 2. Проблема

### Рынок строительных услуг сломан

- **73% заказчиков** сталкивались с недобросовестными исполнителями (исследование НАФИ, 2024)
- **Нет прозрачности**: невозможно объективно оценить качество мастера до начала работ
- **Нет финансовых гарантий**: предоплата уходит безвозвратно при срыве сроков
- **Фрагментация**: поиск мастера через «сарафанное радио», доски объявлений, разрозненные сайты

**Типичные боли заказчика:**

| Проблема | Частота |
|----------|---------|
| Срыв сроков | 68% |
| Низкое качество работ | 54% |
| Завышение стоимости в процессе | 47% |
| Невозможность найти мастера | 39% |

**Типичные боли мастера:**

- Нестабильный поток заказов
- Проблемы с получением оплаты
- Высокая конкуренция с демпингующими исполнителями
- Отсутствие инструментов для управления репутацией

---

## Слайд 3. Решение

### МастерОК — платформа нового поколения

**Три ключевых инновации:**

1. **AI-подбор специалистов** — алгоритм скоринга анализирует 6 факторов и 40 весовых параметров для идеального матчинга заказчик-мастер
2. **Milestone-эскроу** — разбивка оплаты на этапы с промежуточной приёмкой. Деньги заказчика защищены до подтверждения каждого этапа
3. **Трекинг прогресса в реальном времени** — заказчик видит ход работ, фотоотчёты, уведомления о каждом шаге

**Результат:** доверие, прозрачность и гарантия для обеих сторон.

## Слайд 6. Как это работает

### Путь пользователя

```
Заказчик создаёт заказ
        |
   AI подбирает топ-10 мастеров
        |
   Мастер откликается, обсуждает детали
        |
   Согласование milestones и бюджета
        |
   Milestone-оплата через эскроу
        |
   Выполнение работ + трекинг прогресса
        |
   Приёмка каждого этапа заказчиком
        |
   Автоматическое перечисление средств мастеру
        |
   Отзыв и рейтинг
```

**Среднее время от заказа до начала работ: 2-4 часа** (vs. 3-7 дней через другие площадки).

## Слайд 9. Технологии

### Технологический стек

**Frontend:**
- Next.js 14 (SSR + клиентские компоненты)
- TypeScript, Tailwind CSS, Framer Motion
- PWA (Progressive Web App) + TWA (Trusted Web Activity)

**Backend:**
- FastAPI (async Python), SQLAlchemy
- PostgreSQL, WebSocket real-time
- JWT авторизация

**AI-алгоритм скоринга:**
- 6 факторов матчинга: специализация, рейтинг, опыт, локация, загруженность, стоимость
- 40 весовых параметров
- Обучение на данных завершённых заказов

**Инфраструктура:**
- Docker Compose (PostgreSQL + Backend + Landing + Nginx)
- YooKassa эскроу-платежи
- Push-уведомления (Web Push API)

## Слайд 11. Unit-экономика

### Ключевые показатели

| Метрика | Значение |
|---------|----------|
| **CAC** (стоимость привлечения клиента) | 1 500 руб. |
| **LTV** (пожизненная ценность клиента) | 38 000 руб. |
| **LTV / CAC** | **25x** |
| **Payback period** | 2 месяца |

**Расчёт LTV:**
- Среднее количество заказов в год: 3
- Средняя комиссия с заказа: 12 750 руб.
- Средний срок жизни клиента: 1 год (консервативно)
- LTV = 3 x 12 750 = 38 250 руб.

**Каналы привлечения:**
- VK Ads (таргетированная реклама): CAC ~1 200 руб.
- Яндекс.Директ (контекстная реклама): CAC ~1 800 руб.
- SEO (органический трафик): CAC ~500 руб. (с 6 месяца)
- Средневзвешенный CAC: 1 500 руб.
