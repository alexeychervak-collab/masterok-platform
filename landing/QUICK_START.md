```bash

npm install --force
npm run dev
```
Откройте: **http://localhost:3000**

## 📦 Если что-то не работает

### 1. Ошибка "command not found"
```bash
# Установите Node.js 18+
# https://nodejs.org/
```

### 2. Ошибка при сборке
```bash
cd D:\masterok\landing
rm -rf node_modules
rm package-lock.json
npm install --force
npm run build
```

### 3. Порт 3000 занят
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# Linux/Mac
lsof -ti:3000 | xargs kill -9
```
