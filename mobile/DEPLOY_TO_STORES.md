## 🤖 GOOGLE PLAY (ANDROID)

### Предварительные требования

1. ✅ Аккаунт Google Play Developer ($25 один раз)
2. ✅ Собранный AAB файл
3. ✅ Иконка приложения (512x512px)
4. ✅ Скриншоты (минимум 2)
5. ✅ Описание приложения

### Шаг 1: Сборка App Bundle

```bash
cd mobile

# Полная сборка
BUILD_RELEASE.bat

# Или только AAB
flutter build appbundle --release
```

**Результат:** `build\app\outputs\bundle\release\app-release.aab`

### Шаг 2: Создание приложения в консоли

1. Открыть https://play.google.com/console
2. "Создать приложение"
3. Заполнить:
   - **Название:** STROYKA - Строительные услуги
   - **Язык по умолчанию:** Русский
   - **Тип:** Приложение
   - **Бесплатное/Платное:** Бесплатное

## 🍎 APP STORE (iOS / TESTFLIGHT)

### Предварительные требования

1. ✅ Apple Developer Account ($99/год)
2. ✅ Mac с Xcode 15+
3. ✅ Сертификаты и профили подготовлены
4. ✅ Собранный IPA файл

### Шаг 1: Настройка Xcode

```bash
cd mobile/ios
pod install
open Runner.xcworkspace
```

#### В Xcode:

1. **Signing & Capabilities**
   - Team: выбрать свою команду
   - Bundle Identifier: `com.stroyka.app`
   - Signing: Automatically manage signing

2. **General**
   - Display Name: `STROYKA`
   - Version: `1.0.0`
   - Build: `1`

3. **Build Settings**
   - iOS Deployment Target: `15.0`
   - Swift Language Version: `5.0`

### Шаг 2: Создание App в App Store Connect

1. Открыть https://appstoreconnect.apple.com
2. "Мои приложения" → "+" → "Новое приложение"
3. Заполнить:
   - **Платформа:** iOS
   - **Название:** STROYKA
   - **Основной язык:** Русский
   - **Bundle ID:** com.stroyka.app
   - **SKU:** stroyka-ios

### Шаг 4: Сборка IPA для TestFlight

#### В Xcode:

1. **Product** → **Scheme** → **Edit Scheme**
2. Выбрать **Release** для Run

3. **Product** → **Archive**
4. Дождаться окончания (10-15 минут)

5. В Organizer:
   - Выбрать последний архив
   - **Distribute App**
   - **App Store Connect**
   - **Upload**
   - Дождаться загрузки

### Шаг 5: Настройка TestFlight

1. В App Store Connect → TestFlight
2. Дождаться обработки билда (10-30 минут)
3. "Управление соответствием" → заполнить опрос
4. "Информация для тестировщиков":

```
Спасибо за тестирование STROYKA!

Что тестировать:
1. Регистрация / Вход
2. Поиск специалистов
3. Создание заказа
4. Чат
5. Профиль

Известные проблемы:
- Платежи работают в тестовом режиме

Обратная связь: dev@stroyka.ru
```

### Шаг 6: Добавление тестировщиков

#### Внутреннее тестирование:

1. "Внутреннее тестирование"
2. "+" → Добавить тестировщиков
3. Ввести email (до 100 адресов)
4. Тестировщики получат приглашение

#### Внешнее тестирование:

1. "Внешнее тестирование"
2. Создать группу
3. Добавить тестировщиков (до 10,000)
4. Отправить на проверку Apple (24-48 часов)

### Шаг 7: Тестирование

Тестировщики должны:
1. Установить **TestFlight** из App Store
2. Открыть письмо-приглашение
3. Нажать "View in TestFlight"
4. Установить и тестировать

### Шаг 8: Отправка на модерацию App Store

После успешного тестирования:

1. В App Store Connect → "Мое приложение"
2. Выбрать билд из TestFlight
3. Заполнить оставшиеся поля
4. "Отправить на проверку"
5. Ожидать модерацию (1-3 дня)

## 🔧 PRODUCTION НАСТРОЙКИ

### Firebase

Переключить на production:

```dart
// lib/core/services/firebase_service.dart
const bool kReleaseMode = true; // Изменить на true
```

### YooKassa

Заменить тестовые ключи:

```dart
// lib/core/services/yookassa_service.dart
static const String _clientApplicationKey = 'live_YOUR_REAL_KEY';
static const String _shopId = 'YOUR_REAL_SHOP_ID';
```

```python
# backend/.env
YOOKASSA_SHOP_ID=real_shop_id
YOOKASSA_SECRET_KEY=live_REAL_SECRET_KEY
```

### API URL

```dart
// lib/core/network/api_client.dart
static const String baseUrl = 'https://api.stroyka.ru'; // Production URL
```
