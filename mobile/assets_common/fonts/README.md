# SF UI Fonts

Скопируйте шрифты San Francisco из:
```
D:\Пиица\app\assets_common\fonts\SanFrancisco\
```

В эту папку:
```
d:\yodo\mobile\assets_common\fonts\SanFrancisco\
```

Или скачайте с официального сайта Apple.
