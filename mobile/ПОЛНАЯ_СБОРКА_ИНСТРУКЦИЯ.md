# 🏗️ МастерОК - ПОЛНАЯ ИНСТРУКЦИЯ ПО СБОРКЕ

## ⚡ БЫСТРАЯ СБОРКА ANDROID APK

### Способ 1: Автоматическая сборка (БЕЗ ПОДПИСИ)

```bash
cd mobile
flutter build apk --release
```

**Результат:** `build/app/outputs/flutter-apk/app-release.apk` (~45 MB)

### Способ 2: Split APK (меньший размер)

```bash
cd mobile
flutter build apk --release --split-per-abi
```

**Результаты:**
- `app-armeabi-v7a-release.apk` (~40 MB) - старые устройства
- `app-arm64-v8a-release.apk` (~45 MB) - новые устройства ⭐
- `app-x86_64-release.apk` (~48 MB) - эмуляторы

### Способ 3: App Bundle для Google Play

```bash
cd mobile
flutter build appbundle --release
```

**Результат:** `build/app/outputs/bundle/release/app-release.aab` (~35 MB)

## 🔐 ПОДПИСАННАЯ СБОРКА (ДЛЯ GOOGLE PLAY)

### Шаг 1: Создать Keystore

**Найти keytool:**
```bash
# Проверить где установлена Java
where java

# Обычно keytool в той же папке
# C:\Program Files\Java\jdk-XX\bin\keytool.exe
```

**Создать keystore:**
```bash
"C:\Program Files\Java\jdk-21\bin\keytool.exe" -genkey -v ^
  -storetype JKS ^
  -keyalg RSA ^
  -keysize 2048 ^
  -validity 10000 ^
  -storepass masterok123456 ^
  -keypass masterok123456 ^
  -alias masterok-key ^
  -keystore android\app\masterok-keystore.jks ^
  -dname "CN=МастерОК, OU=Dev, O=МастерОК, L=Moscow, ST=Moscow, C=RU"
```

**Или используйте bat файл:**
```bash
cd android
create-keystore.bat
```

### Шаг 2: Проверить key.properties

Файл `android/app/key.properties` должен существовать:
```properties
storePassword=masterok123456
keyPassword=masterok123456
keyAlias=masterok-key
storeFile=masterok-keystore.jks
```

### Шаг 3: Собрать подписанный APK

```bash
flutter build apk --release --split-per-abi
```

## 🍎 ПОЛНАЯ СБОРКА iOS

### Требования

- ✅ Mac с macOS 12+
- ✅ Xcode 15+  
- ✅ CocoaPods
- ✅ Apple Developer Account ($99/год)

### Шаг 1: Установить CocoaPods

```bash
sudo gem install cocoapods
```

### Шаг 2: Установить Pods

```bash
cd ios
pod install
```

⏱️ **Первый раз:** 10-15 минут

### Шаг 3: Открыть в Xcode

```bash
open Runner.xcworkspace
```

⚠️ **ВАЖНО:** Открывать `.xcworkspace`, НЕ `.xcodeproj`!

### Шаг 4: Настроить Signing

В Xcode:

1. Выбрать проект **Runner** в Navigator (слева)
2. Target **Runner** → **Signing & Capabilities**
3. **Team:** Выбрать свою команду
4. **Bundle Identifier:** `com.masterok.app`
5. ✅ **Automatically manage signing**

### Шаг 5: Выбрать схему Release

```
Product → Scheme → Edit Scheme
  └─ Run → Build Configuration → Release
```

### Шаг 6: Archive

```
Product → Archive
```

⏱️ **Время сборки:** 15-20 минут

### Шаг 7: Distribute

После успешного Archive:

```
Window → Organizer → Archives
  └─ Последний archive → Distribute App
      └─ App Store Connect
          └─ Upload
```

### Шаг 8: TestFlight

1. Открыть https://appstoreconnect.apple.com
2. Мои приложения → МастерОК → TestFlight
3. Дождаться обработки (10-30 минут)
4. Управление соответствием → Заполнить
5. Добавить тестировщиков

## 🔧 ЕСЛИ ВОЗНИКЛИ ПРОБЛЕМЫ

### Android: "Execution failed for task ':app:lintVitalAnalyzeRelease'"

```bash
# Отключить lint для Release
flutter build apk --release --no-shrink
```

### Android: "Keystore file not found"

```bash
# Создать keystore
cd android
create-keystore.bat
```

### iOS: "Command PhaseScriptExecution failed"

```bash
cd ios
rm -rf Pods Podfile.lock
pod install
```

### iOS: "No valid code signing"

```
Xcode → Preferences → Accounts
  → Добавить Apple ID
  → Download Manual Profiles
```

### Flutter: "Version solving failed"

```bash
flutter clean
flutter pub get
```
